# BITACORA-P5
## SOLEMNE 1
INSPO: El referente que yo elegi fue el estilo **Bauhaus**
### Proceso Creativo:
Parti dibujando las figuras en una hoja 25X25 despues pase mi dibujo a ilustrator en una hoja de 500mm x 500mm ya que al empezar a enumerar las casillas de la
hoja(0,5mm de 10 en 10)  con una tecnica que me enseño una compañera me di cuenta que algunas lineas no estaban bien rectas entonces se me estaba dificultando pasarlas a P5, 
a si que utilizando "AI" pase el dibujo ,asi tambien agregando algunas figuras ya que algunas figuras del dibujo original no se podian pasar de manera exacta a la plataforma de ilustrator , 
y utilizando  la herramienta de pluma me daba las cordenadas X,Y de una manera no tan  exacta pero se acercaba mucho al resultado que yo esperaba entonces me facilito mucho el procedimiento 
de tener X,Y y en algunos casos el diametro y tambien utilizando la herramienta de la grilla.

Despues decidi buscar la paleta de colores de bauhaus y decidi usar los mismos codigos de colores pasandolos a la pagina web de HTML COLOR CODE , teniendo todo eso 
empece a pasar todo a P5 con los codigos enseñados en la clase y usando como apoyo el PPT , se me dificulto mucho el proceso de entender los semicirculos "ARC()"  ya que 
se me habia olvidado poner el programa inicial "angleMode(DEGREES)". No fue un trabajo tan dificil si no que requeria mucho tiempo y tambien presicion para las cordenadas 
y que tambien quede todo de buena manera y encaje todo
